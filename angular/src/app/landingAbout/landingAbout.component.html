The NIST Public Data Repository (PDR) is a key component of NIST research data infrastructure supporting public dissemination through open data sharing and access to an evolving collection of NIST public datasets. The main goals aim to communicate the results of NIST research to foster reproducibility, interoperability and discovery of scientific information for the community at large. Each PDR record includes descriptive information about a public data resource and corresponds to an entry in the NIST Enterprise Data Inventory, a fully compliant government open data standard listing.

The PDR is designed as a trusted repository, one which complies with long term data preservation and publication standards. Each record is viewable through a unique web “landing page” which includes user-friendly and readable views of the record description and metadata, high level terms which detail the nature of the record dataset. Additionally through the PDR landing page interface, users have access to underlying data files, Fair Use policy and citation information, and a suite of related details including linkage to the machine readable version of the record. The top banner “Search” feature navigates to the NIST Data Discovery portal where users may perform queries and explore the PDR datasets of interest.

In summary, the NIST PDR implements the following features based on FAIR data principles:

  • Search and Navigation for NIST public datasets
  • Citable web landing pages for published datasets with Digital Object Identifiers (DOIs)
  • Find author related publications
  • Access methods to retrieve data files
  • Interoperability through machine readable standard formats of datasets
  • Long term digital preservation
